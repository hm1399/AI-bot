# 连接指南

## 🔌 为什么显示 "Disconnected"（断开连接）？

当您首次打开应用时，会显示 "Disconnected" 是**正常现象**，因为：

1. ✅ 应用还没有连接到您的 AI-Bot 设备
2. ✅ 您需要先配置服务器连接信息
3. ✅ 这是为了保护您的隐私和安全

## 📱 正确的使用流程

### 第一步：连接页面

当您打开应用时，会自动跳转到**连接页面** (`/connect`)：

```
应用启动
   ↓
自动跳转到连接页面
   ↓
选择连接方式
```

### 第二步：选择连接方式

#### 方式 1：自动发现（推荐）

1. 点击 **"Scan Network"**（扫描网络）按钮
2. 应用会自动搜索局域网内的 AI-Bot 设备
3. 找到设备后，点击设备名称即可连接

#### 方式 2：手动输入

1. 在 "Server IP / Hostname" 输入框输入服务器 IP 地址
   - 例如：`192.168.1.100` 或 `localhost`
2. 在 "Port" 输入框输入端口号
   - 默认：`8000`
3. 点击 **"Connect"**（连接）按钮

### 第三步：连接成功

连接成功后：

1. ✅ 自动跳转到主应用界面 (`/app`)
2. ✅ 顶部状态栏变为**绿色**，显示 "Connected"（已连接）
3. ✅ 可以开始使用所有功能

## 🔄 应用路由结构

```
/                    → 自动重定向到 /connect
/connect            → 连接页面（首次使用）
/app                → 主页面（需要先连接）
  ├─ /app           → 首页（设备状态）
  ├─ /app/chat      → 对话页面
  ├─ /app/tasks     → 任务和日程
  └─ /app/settings  → 设置页面
```

## 🎯 状态栏说明

### 🟢 绿色 - "Connected"（已连接）
- WebSocket 连接正常
- 可以实时收发消息
- 所有功能可用

### 🔴 红色 - "Disconnected"（断开连接）
- 未连接到服务器
- 或者连接已断开
- 需要重新连接

## 🛠 连接前的准备工作

### 1. 启动后端服务器

确保您的 Python 后端服务器正在运行：

```bash
# 在后端项目目录
python main.py
# 或
uvicorn main:app --host 0.0.0.0 --port 8000
```

### 2. 检查网络连接

- ✅ 设备和电脑在同一个 WiFi 网络
- ✅ 防火墙允许端口 8000
- ✅ 服务器 IP 地址正确

### 3. 获取服务器 IP

**Windows:**
```cmd
ipconfig
# 查找 "IPv4 地址"
```

**Mac/Linux:**
```bash
ifconfig
# 或
ip addr show
```

**Linux 快速获取:**
```bash
hostname -I
```

## 🔍 常见问题

### 问：为什么扫描不到设备？

**答：** 可能的原因：
1. 后端服务器未启动
2. 设备不在同一网络
3. 防火墙阻止了网络发现
4. 使用手动连接作为替代方案

### 问：连接后立即断开怎么办？

**答：** 检查：
1. 后端服务器是否正常运行
2. WebSocket 端点是否正确（`ws://IP:8000/ws/app`）
3. 查看浏览器控制台（F12）的错误信息
4. 检查后端日志

### 问：可以在手机上使用吗？

**答：** 可以！
1. 确保手机和服务器在同一 WiFi
2. 在手机浏览器输入服务器地址
3. 输入服务器 IP 和端口连接
4. 应用是响应式设计，完美支持移动端

### 问：连接信息会保存吗？

**答：** 会的！
- 连接成功后，服务器地址和端口会保存在本地
- 下次打开应用会尝试自动重连
- 但首次使用需要手动配置

## 📝 示例连接配置

### 本地测试
```
Server IP: localhost
Port: 8000
```

### 局域网设备
```
Server IP: 192.168.1.100
Port: 8000
```

### 自定义端口
```
Server IP: 192.168.1.100
Port: 5000
```

## 🎉 连接成功后可以做什么？

1. **查看设备状态** - 电池、WiFi 信号、当前状态
2. **快速操作** - 静音、LED 控制、重启设备
3. **对话聊天** - 与 AI 助手对话
4. **管理任务** - 创建、完成、删除任务
5. **日程安排** - 添加和管理事件
6. **自定义设置** - 配置 LLM、语音、LED 等

## 💡 小贴士

1. **首次使用建议使用手动连接** - 更快更可靠
2. **保存常用 IP** - 记住您的服务器 IP 地址
3. **检查连接状态** - 顶部状态栏随时查看
4. **断线会自动重连** - 5 秒后自动尝试重连
5. **Settings 可以断开连接** - 需要更换设备时使用
