# AI-bot - 桌面AI助手

一款集成语音交互、电脑控制、智能待办、AI陪伴等功能的桌面AI助手，支持硬件设备和手机App远程控制。

## 项目简介

桌面AI助手是一个软硬件结合的智能助手项目，通过语音或文字与用户交互，可以控制电脑文件和应用、管理日程待办、提供AI陪伴对话，并支持多种物理交互方式。

## 核心功能

### 1. 电脑文件管理与应用控制（MVP）
- 文件操作：创建、删除、移动、重命名、搜索文件/文件夹
- 应用程序控制：打开、关闭、切换应用
- 系统操作：调节音量、亮度、截图、锁屏等

### 2. 手机App远程控制
- 发送文字/语音指令给AI
- 查看AI执行状态和反馈
- 远程触发预设任务
- 接收推送通知

### 3. 语音交互（MVP）
- 语音唤醒词
- 语音指令识别
- 语音回复
- 中英文支持

### 4. 桌面摆件模式（待机状态）（MVP）
- 时间显示
- 天气信息
- 日历/提醒事项
- 可自定义显示内容

### 5. AI智能待办（有时间就做）
- 语音添加任务（如"提醒我下午3点开会"）
- 智能解析时间、地点、优先级
- 多种提醒方式：语音、震动、灯光、手机推送、屏幕弹窗
- 语音查询待办列表

### 6. AI人格系统（有时间就做）
- 多种预设人格：温柔治愈型、傲娇型、毒舌型、老友型、管家型、二次元型
- 支持自定义人格
- 陪伴对话（闲聊模式）
- 情绪回应与话题记忆

### 7. 智能场景模式（有时间就做）
- **专注模式**：开启勿扰、关闭社交软件、播放白噪音
- **下班模式**：关闭工作软件、整理文件、生成工作总结
- **会议模式**：系统静音、会议录音、实时转写、AI会议总结

### 8. 物理交互（有时间就做）
- 摇一摇抽签（运势、决策、名言）
- 拍一拍交互（1次允许/2次拒绝/3次打断）
- 硬件按钮切换纯聊天模式

### 9. AI日历（有时间就做）
- 语音管理日程（添加/修改/删除）
- 智能提醒（早晨播报、活动前提醒）
- 日历同步（Apple/Google/Outlook）
