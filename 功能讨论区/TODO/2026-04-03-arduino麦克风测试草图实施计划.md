# Arduino 麦克风测试草图 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `firmware/arduino` 下新增一个可直接烧录到当前硬件板的 `INMP441` 麦克风诊断草图，验证 `IO14/IO15/IO16` 链路和声道活跃情况。

**Architecture:** 新增独立测试目录 `test12_mic_diag`，保持不影响现有 `test3`。草图继续使用仓库里已验证过的 `32-bit I2S slot` 读取方式，对 `slot0` 和 `slot1` 分别统计平均绝对值与峰峰值，再根据更活跃的一侧输出 ASCII 音量条、状态提示和串口命令诊断。

**Tech Stack:** Arduino C++、ESP32 Arduino Core、`driver/i2s.h`、USB CDC 串口
