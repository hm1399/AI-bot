# 任务

将 `app/` 调整为仅面向桌面端的 Flutter 工程，目标平台为 `Windows + macOS + Linux`。

# 范围

- 以 `app/` 为唯一主前端工程
- 补齐桌面平台工程文件
- 修复当前会影响桌面运行、分析或构建的错误
- 无关目录先保留，但不得影响主工程正常运行

# 执行步骤

1. 检查 `app/` 当前平台支持、依赖和报错。
2. 生成或补齐 Windows、macOS、Linux 桌面工程。
3. 修复桌面端相关代码、配置和分析错误。
4. 验证 `flutter analyze`、`flutter test`，以及当前环境可执行的平台构建。
5. 用 git 提交本次任务修改。
