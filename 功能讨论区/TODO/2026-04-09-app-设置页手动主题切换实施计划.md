# app 设置页手动主题切换实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 Flutter 前端增加设置页手动主题切换，支持 `浅色 / 深色` 两档，并在本机持久化保存。

**Architecture:** 主题模式属于前端本地 UI 偏好，不进入后端设置接口。实现上通过单独的本地存储 service 保存 theme preference，在 app 级状态中维护当前 `ThemeMode`，`main.dart` 读取该状态驱动全局主题，设置页新增 `Appearance` 区块直接更新全局模式。

**Tech Stack:** Flutter, Material 3, hooks_riverpod, shared_preferences
