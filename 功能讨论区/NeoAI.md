# NeoAI 项目调研

## 项目简介

NeoAI 是一款开源的AI电脑控制工具，让用户无需编写代码，只需使用自然语言指令即可控制电脑。

**项目地址**: https://github.com/TheD0ubleC/NeoAI

**核心理念**: 无需编写代码，只需会打字！让AI用简单的自然语言指令接管你的电脑。

## 核心功能

1. **文件管理** - 通过自然语言指令管理文件和文件夹
2. **任务自动化** - 处理重复性任务，提高工作效率
3. **定时操作** - 设定特定时间执行任务（如定时重启、关机）
4. **跨平台设备控制** - 支持多平台操作
5. **内置安全保护** - 确保操作安全

## 安装方法

```bash
# 1. 克隆项目
git clone https://github.com/TheD0ubleC/NeoAI.git

# 2. 进入目录
cd NeoAI

# 3. 安装依赖
pip install -r requirements.txt

# 4. 进入src目录并运行
cd src
python web-ui.py
```

启动后在浏览器打开对应端口即可使用。
